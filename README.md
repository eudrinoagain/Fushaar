# Self Host

### To self host **Fushaar** you need to follow the steps:

## 1. Clone the repo

## 2. run `npm install`

## 3. change the credentials in the `.env` file

## 4. setup slash commands in your server (**[Slash Commands GUI](https://slash-commands-gui.androz2091.fr/)**)

## 4. run `npm run start`
